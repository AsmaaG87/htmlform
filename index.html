HTML Form Cheat Sheet

HTML Essential Form Elements
Element Description Example
<form> Creates an HTML form for user input. <form action="/submit" method="POST/GET"></form>
<label> Defines a label for an input element. <label for="name">Name:</label>
<input> Creates an input field (text, password, checkbox, radio, etc.) <input type="text" name="name" id="name">
<button> Creates a clickable button. <button type="submit">Submit</button>
<textarea> Creates a multi-line text input field. <textarea name="message" rows="4"></textarea>
<select> Creates a drop-down menu. <select name="options"><option value="1">Option1</option></select>
<option> Defines options within a<select>drop-down. <option value="1">Option 1</option>
<fieldset> Groups related form elements. <fieldset><legend>Personal Info</legend></fieldset>
<legend> Defines a title for a<fieldset> <legend>Personal Info</legend>
